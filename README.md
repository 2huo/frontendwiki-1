# 前端维基 FrontendWiki

前端知识日新月异，需要学习的知识点很多很杂，程序员们都希望掌握优质的学习资源，然而大部分官方文档都不够简单清晰，对新手不友好，这时候就需要各方共同努力，筛选出好的资源。

此站点通过建立一个**树状知识结构**组织起来的前端知识学习资料**链接索引**，对知识领域进行**分档、标签化**归类，用户可通过**点赞、评论**等方式推荐，以达到筛选出优质学习资源的目的。<u>汇总优质资源，避免二次创作</u>。

## 概述
